# 💸 Vibe Money --- Desafio Vibe Coding

## 🚀 Minha Ideia

A proposta do **Vibe Money** é ser um aplicativo simples, intuitivo e
acessível para organização de finanças pessoais.

A ideia central foi construir uma solução clara e objetiva, permitindo
que o usuário controle receitas e despesas sem esforço excessivo ou
complexidade.

O foco está na experiência: poucos passos, visual limpo e interação
natural.

------------------------------------------------------------------------

## 🧠 Minha Experiência com Vibe Coding

Durante o desafio, percebi que **Vibe Coding não é sobre programar linha
por linha**, mas sobre saber comunicar sua intenção com clareza.

Utilizei o Copilot para estruturar meu prompt inicial (PRD) e garantir
que ele estivesse claro, organizado e objetivo. Em seguida, levei esse
prompt ao Lovable para iniciar a construção conceitual do app.

Após a primeira interação com a IA, percebi oportunidades de melhoria
--- principalmente em relação à experiência do usuário e refinamento da
lógica de funcionamento. Então, complementei as instruções para deixar
minha intenção ainda mais clara.

### ✅ O que funcionou bem

-   Estruturar o PRD antes de pedir qualquer geração.
-   Ser específico nas funcionalidades obrigatórias.
-   Definir identidade visual e comportamento esperado.

### ⚠️ O que precisei ajustar

-   Refinar instruções para melhorar a clareza do fluxo.
-   Especificar melhor como as atualizações automáticas deveriam
    acontecer.
-   Reforçar a simplicidade como prioridade do produto.

### 📚 O que aprendi

-   Quanto mais clara a intenção, melhor o resultado da IA.
-   A IA responde melhor quando recebe contexto e objetivo.
-   Pequenos ajustes no prompt geram grandes diferenças no resultado.

------------------------------------------------------------------------

## 🤝 Minha Interação com a IA

1.  Criei o prompt base utilizando o Copilot.
2.  Levei o prompt ao Lovable para gerar o plano inicial do MVP.
3.  Analisei o retorno.
4.  Identifiquei melhorias necessárias.
5.  Ajustei o prompt com instruções complementares.
6.  Direcionei a IA para melhorar fluxo, lógica e experiência.

Essa interação mostrou que a IA funciona como uma parceira criativa ---
mas precisa de direção clara.

------------------------------------------------------------------------

## 📌 Meu Prompt Final (PRD)

📌 PROMPT PARA O LOVABLE --- VIBE MONEY Crie um aplicativo chamado Vibe
Money para organização de finanças pessoais. O app deve ser simples,
intuitivo e ideal para usuários iniciantes.

🎯 Funcionalidades obrigatórias

1.  Tela Inicial (Dashboard) • Exibir três cards: ◦ Total de Receitas ◦
    Total de Despesas ◦ Saldo Final (Receitas -- Despesas) • O saldo
    deve aparecer em: ◦ Verde → se for positivo ◦ Vermelho → se for
    negativo • Botão para adicionar nova transação

2.  Tela de Lista de Transações • Lista todas as transações cadastradas
    • Cada item da lista deve mostrar: ◦ Tipo (receita ou despesa) ◦
    Descrição ◦ Valor ◦ Categoria ◦ Data • Identificar receitas com cor
    verde • Identificar despesas com cor vermelha

3.  Tela de Nova Transação Formulário com os seguintes campos
    obrigatórios: • Tipo: receita ou despesa (toggle ou select) •
    Descrição • Valor • Categoria • Data (definir padrão como data
    atual)

Ação ao salvar: • Adicionar a transação à lista • Atualizar totais no
dashboard • Retornar para a tela inicial após salvar

4.  Armazenamento e lógica • Usar armazenamento local para salvar todas
    as transações • Garantir que os dados não sejam perdidos ao fechar o
    app • Criar funções automáticas para: ◦ Somar todas as receitas ◦
    Somar todas as despesas ◦ Calcular o saldo final

🎨 Estilo e identidade visual • Visual moderno e limpo • Paleta
sugerida: tons de verde, branco e preto • Títulos com destaque • Ícones
minimalistas • App com sensação de simplicidade e organização • Nome
Vibe Money no topo da tela principal

Esse é o app base. Se algo faltar, quero pedir ajustes depois.

------------------------------------------------------------------------

## 💬 Conclusão

O desafio mostrou que desenvolver com IA é sobre **clareza, intenção e
estratégia**.

Mesmo sem escrever código, consegui estruturar um conceito funcional,
com lógica definida, identidade visual e foco no usuário.

Vibe Coding me ensinou que a IA não substitui o pensamento --- ela
amplia quando sabemos guiá-la.
